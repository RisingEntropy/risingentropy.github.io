# AI(Aritficial Intelligence I)
可以从哲♂学层面揭示人类本质的AI

```cpp
# include <iostream>
# include <string>
using namespace std;
int main(){
	cout<<"Welcome to Artificial Intelligence I"<<endl;
	cout<<"Let's chat"<<endl;
	while(true){
		cout<<"You:";
		string input;getline(cin,input);
		cout<<"Artificial Intelligence I:"<<input<<"♂"<<endl;
	}
	return 0;
}
```
