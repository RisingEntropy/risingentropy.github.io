【偏序，cdq】陌上花开

Haoyu Deng
26 May 2019

题面

有 n 个元素，第 i 个元素有 ai bi ci 三个属性，设 f(i) 表示满足 aj≤ai 且 bj≤bi 且 cj≤ci 的 j 的数量。

对于 d∈[0,n) ，求 f(i)=d 的 i 的数量。

输入 第一行两个整数 n 、k，分别表示元素数量和最大属性值。

之后 n 行，每行三个整数 ai bi ci分别表示三个属性值。

输出 输出 n行，第 d+1 行表示 f(i)=d 的 i的数量。

样例输入

10 3
3 3 3
2 3 3
2 3 1
3 1 1
3 1 2
1 3 1
1 1 2
1 2 2
1 3 2
1 2 1

样例输出

3
1
3
0
1
0
1
0
0
1

提示 1≤n≤100000,1≤k≤200000

解答

留个坑，来不及写了。滚回学校关起
